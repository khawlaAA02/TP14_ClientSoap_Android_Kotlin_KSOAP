## TP14_ClientSoap_Android_Kotlin_KSOAP

Ce TP montre comment intégrer Android avec un Web Service SOAP en utilisant KSOAP2, et comment gérer une interface dynamique et moderne avec RecyclerView + Material Design + Coroutines.
Il illustre une architecture propre, évolutive et proche des applications professionnelles.
